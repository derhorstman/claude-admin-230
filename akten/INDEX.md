## Status-Werte

- **Neu** = Gerade angelegt
- **In Bearbeitung** = Aktiv dran
- **Warte auf Antwort** = Mail gesendet, warte auf Reaktion
- **Erledigt** = Abgeschlossen

## Workflow

1. "Neue Akte zu [Thema]" → Claude legt Ticket an
2. "Lies Mail zu [Thema]" → Claude liest und fasst zusammen
3. "Schreib Antwort" → Claude erstellt Mail-Entwurf auf Mac
4. "Ticket erledigt" → Claude schließt ab
